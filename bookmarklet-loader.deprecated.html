Listing Enrichment Bookmarklet

On realestate.com.au? Use the inline bookmarklet instead. REA's Content-Security-Policy silently blocks the loader version below (it injects an external script REA won't allow), so "nothing happens". The inline version carries the whole script in the link itself and works on REA and Domain alike.

This bookmarklet extracts listing details (photo, beds, baths, etc.) from Domain or REA listing pages and saves them to your dashboard. If the listing is already tracked it is enriched in place; if it is not yet in the dashboard, it is added automatically as a new entry (geocoded and Tier-1 scored on the spot).

How to Use

  1. Make sure the local server is running
  2. Open the dashboard and click "Open listing" on any property
  3. When the Domain/REA listing page loads, click the Enrich Listing bookmark
  4. A popup will confirm what was saved
  5. Refresh the dashboard to see the updated listing with photo

What Gets Extracted

  • Cover photo URL
  • Beds, baths, parking
  • Property type
  • Price guide
  • Description
  • Address, suburb and postcode (from the listing heading, with the URL slug as a fallback)
  • Direct listing URL (replaces search URL)

When the page isn't already tracked, these same fields are used to mint a brand-new dashboard entry. The popup tells you whether the listing was matched to an existing one or added as new.
