Enrich Listing — bookmarklet

This is the single, canonical install page for the dashboard bookmarklet. It supersedes the earlier two pages, which are kept only for reference as bookmarklet-loader.deprecated.html and bookmarklet-inline.deprecated.html.

Open the live version: Go to the bookmarklet →
With your local server running (python scripts/serve.py), that page shows the draggable ⭐ Enrich Listing button, generated fresh from the current enrich-bookmarklet.js so it always carries every upgrade (inline / CSP-proof so it works on realestate.com.au and Domain alike; address, beds/baths/parking, and price extraction for both sites).
Why a link and not the button here? The button bakes the whole script into the bookmark, so it must be regenerated whenever enrich-bookmarklet.js changes. serve.py does that and rewrites this very file with the up-to-date button each time you open /bookmarklet — so after you have visited it once with the server running, this page will itself show the live button. Until then, use the link above.

How to use

  1. Run python scripts\serve.py (required — the bookmarklet saves by POSTing to http://localhost:8777).
  2. Open http://localhost:8777/bookmarklet and drag the ⭐ Enrich Listing button to your bookmarks bar. Delete any older "Enrich Listing" bookmark so you don't click the wrong one.
  3. On a Domain or realestate.com.au listing, click the bookmark. A popup confirms what was saved (and adds the listing as a new entry if it isn't already tracked).
  4. Reload the dashboard to see the updated listing.
